<section class="hero">
  <h1>Berkeley Interim Housing: Police Call Analysis</h1>
  <p class="lead">
    Did non-traffic police calls increase near four University Avenue motel conversions after they opened as interim housing?
    This site presents a before-and-after analysis using public Berkeley Police data.
  </p>
  <p class="scope-note">
    This analysis measures association and timing patterns.
    It does not establish causation.
  </p>
</section>

<section class="about-data">
  <h2>Data and Methods</h2>
  <p>
    Analysis uses Berkeley's public calls-for-service data.
    We focused on non-traffic calls — disturbances, welfare checks, theft, and similar public-safety calls —
    and compared 12 months before and after each site's opening date.
